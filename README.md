# Mindro Monorepo

Este repositório centraliza os projetos do Mindro como monorepo.

## Estrutura

- `admin_console_mindro_learning/` – painel administrativo
- `api_mindro_learning/api/` – backend NestJS
- `app_mindro_learning/` – app React Native / Expo

## Scripts

- `npm run admin:start`
- `npm run admin:build`
- `npm run api:start:dev`
- `npm run api:build`
- `npm run app:start`
- `npm run app:android`
- `npm run app:ios`

## Observações

Os projetos continuam independentes internamente, mas passam a ser gerenciados a partir da raiz do monorepo.
